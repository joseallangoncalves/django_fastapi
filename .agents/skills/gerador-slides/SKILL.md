# Gerador de Slides 📊

Esta skill transforma conteúdos densos em roteiros estratégicos para apresentações. O foco é **impacto visual, brevidade e retenção de audiência**.

## Princípios de Design de Conteúdo
1. **Regra de Ouro:** Menos texto é mais. Cada slide deve conter no máximo 3 a 5 tópicos (bullets).
2. **Dados em Evidência:** Números, estatísticas e KPIs devem ser destacados de forma que não se percam no texto.
3. **Padrão Visual:** Sugerir sempre referências claras de imagens ou vídeos para o fundo ou suporte do slide.
4. **Volume:** O padrão é gerar entre **10 a 15 slides**, permitindo uma narrativa completa.

## Estrutura do Roteiro (Markdown)
O roteiro deve seguir este formato para cada slide:

---
### [SLIDE N: Nome do Slide]
- **Título Visual:** (Um título curto e impactante)
- **Conteúdo em Tópicos:**
    - Tópico 1 (Curto)
    - Tópico 2 (Curto)
    - Tópico 3 (Curto)
- **Dados Destacados:** (Ex: "35% de aumento", "R$ 1.5M economizados")
- **Sugestão Visual:** 
    - **Imagem:** (Prompt/Descrição para gerador como Midjourney/DALL-E)
    - **Vídeo:** (Referência de banco de vídeos como Pexels/Shutterstock)
---

## Exemplo de Aplicação
**Entrada:** Um artigo sobre o mercado de IA em 2026.
**Saída:**
---
### [SLIDE 1: Abertura]
- **Título Visual:** O Amanhã é Agora: IA em 2026
- **Conteúdo em Tópicos:**
    - A transição do experimental para o essencial.
    - O impacto na produtividade global.
- **Sugestão Visual:** 
    - **Imagem:** Um escritório futurista onde humanos trabalham em harmonia com hologramas de IA, estilo cyberpunk soft.
    - **Vídeo:** Time-lapse de uma cidade tecnológica com conexões de dados fluindo entre os prédios.
---

## Passo a Passo para o Modelo
1. **Analisar o Documento:** Identificar a tese principal, os 3-5 pilares argumentativos e todos os dados numéricos.
2. **Dividir em Slides:** Criar uma sequência lógica:
    - Slide 1: Gancho/Abertura
    - Slides 2-4: Problema/Contexto
    - Slides 5-10: Solução/Dados Técnicos/Resultados
    - Slides 11-13: Casos de Uso/Próximos Passos
    - Slide 14-15: CTA (Chamada para Ação) / Conclusão
3. **Refinar:** Revisar se algum slide está "poluído" demais e quebrar em dois se necessário.
4. **Sugerir Visuais:** Ser específico nas cores e estilos (ex: "Glassmorphism", "Vibrant colors", "Professional dark mode").
